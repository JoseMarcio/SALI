Bem Vindo

Primeiras conclusões do projeto. Concluido registrar professor e turma. Ambos prontos para testes e descobrimento de bugs. Registro de aluno já em andamento e pronto para ser concluido neste fim de semana (para a fase de testes).

É notável que para se registrar alunos é necessário pesquisar turma. Sendo que uma turma possui vários alunos.

Falta fazer: - Autenticação: tendo em vista que não é somente uma pesquisa para saber se o usuário está presente no banco de dados. Mas é necessário que se tenha um controle para que cada usuário acesse somente o que lhe é permitido.

Também é ideal (ideal não, necessário) fazer a criptografia de senhas do usuário(provavelmente com MD5, já que com essa, não é possivel revertê-la), e construir a recuperação de senhas do usuário. Por onde atravéz da mesma o sistema criará uma senha aleatório e enviará via e-mail para o usuario os dados de acesso.
